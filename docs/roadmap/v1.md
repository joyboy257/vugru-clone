# Roadmap — v1

Target: working end-to-end prototype that a real estate agent can uses.

## What's built

### Auth
- [x] Magic link login (`POST /api/auth/login` → issues JWT)
- [x] Signup (`POST /api/auth/signup` → creates account with 1000 free credits)
- [x] JWT middleware on protected routes
- [x] Logout (`POST /api/auth/logout`)

### Core loop
- [x] Project CRUD (`/api/projects`)
- [x] Photo upload (presigned R2 URL → `POST /api/upload/confirm`)
- [x] Photo delete + reorder (`DELETE /api/photos/[id]`, `POST /api/projects/[id]/reorder`)
- [x] Clip generation request + queue (`POST /api/clips/generate` → BullMQ)
- [x] Clip status polling (`GET /api/clips/[id]`)
- [x] Clip download (`GET /api/clips/[id]/download`)
- [x] GPU worker: Replicate → R2 pipeline (`workers/video-render/`)
- [x] Credit deduction on clip generate (deducts before queue)

### Virtual staging
- [x] Worker (`workers/virtual-stage/`)
- [x] API route (`POST /api/photos/[id]/virtual-stage`)
- [x] Editor UI (style picker, before/after)

### Auto-edit
- [x] Auto-edit creation (`POST /api/auto-edits`)
- [x] Auto-edit list + fetch (`GET /api/auto-edits/[id]`)
- [x] Auto-edit shareable links (`POST /api/auto-edits/[id]/share`)

### Billing
- [x] Stripe checkout (`POST /api/billing/checkout`)
- [x] Stripe webhook handler (`POST /api/webhooks/stripe`)
- [x] Credit balance fetch (`GET /api/billing/credits`)
- [x] Credit history (`GET /api/billing/history`)
- [x] Billing frontend (`/dashboard/billing`)

### Frontend
- [x] Landing page (hero, how-it-works, features, pricing)
- [x] Magic link login + signup pages
- [x] Dashboard (project list, stats)
- [x] Project editor (photo grid with reorder/delete, clip generation with status polling)
- [x] Settings page

---

## What's missing

### Auth
- [ ] **Password reset** — no `POST /api/auth/forgot-password` route, no email sending
- [ ] **Google OAuth** — no `/api/auth/google` route or callback
- [ ] **Email + password login** — signup has no password field, login is magic-link-only
- [ ] **Magic link rate limiting** — no brute-force or spam protection

### Core loop
- [ ] **Verify clip generation end-to-end** — CogVideoX model ID may be wrong, needs testing with real credits
- [ ] **Sky replacement** — UI buttons exist but no `POST /api/photos/[id]/sky-replace` route and no worker
- [ ] **Batch clip generation** — "generate all" button not built, currently one clip at a time
- [ ] **Resolution selector** — backend accepts `resolution` param but no UI for 720p/1080p/4K pricing

### Auto-edit
- [ ] **Title text generation** — `titleText` stored but no AI generation
- [ ] **Auto-edit render pipeline** — clips not assembled into final MP4 (no concat, no music, no title overlay)
- [ ] **AI music generation** — `musicKey` stored but no generation API
- [ ] **Auto-edit status page** — user can create but can't see render progress

### Billing
- [ ] **Invoice history page** — route exists, no frontend at `/dashboard/invoices`
- [ ] **Credit purchase confirmation toast** — webhook fires but no clear UX feedback

### Secondary features
- [ ] **Duplicate project** — `POST /api/projects/[id]/duplicate`
- [ ] **Bulk delete projects** — `POST /api/projects/bulk-delete`
- [ ] **Project thumbnails** — not set automatically from first photo
- [ ] **Email notification** when clip is done (Polling is the workaround for now)
