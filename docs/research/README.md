## How research works

When a question comes up that needs investigation (a tech choice, a provider evaluation, a user flow question), open a new doc here. Name it descriptively: `models.md`, `auth-flows.md`, `pricing-comparison.md`.

When a research doc results in a decision, that decision gets written to the engineering docs or the architecture doc, and this doc stays as the historical record of how we got there.

> Research docs are writable mid-investigation. They don't need to be finished to be useful.
